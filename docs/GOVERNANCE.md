# Gobernanza y Calidad

## 1. Criterios de Calidad

- Groundedness: afirmaciones críticas respaldadas por evidencia.
- Coherencia: consistencia interna de resultados.
- Relevancia: alineación con objetivos de misión.
- No redundancia: minimización sistemática de repeticiones.

## 2. Principios de Integridad

- Incertidumbre explícita cuando la evidencia es débil o insuficiente.
- Procedencia trazable para conclusiones de investigación y análisis.
- Separación clara entre observaciones e inferencias.

## 3. Estándares de Reporte

- Resumen ejecutivo para tomadores de decisión.
- Núcleo técnico para audiencia especialista.
- Transparencia de fuentes y límites de alcance.

## 4. Política de Evolución

- Toda actualización arquitectónica debe incluir racional e impacto esperado.
- Las capacidades nuevas deben preservar garantías de gobernanza existentes.
